This folder contains the following:

Utility = GlobalEnglish library folders
3rdParty = 3rd party library folders
Common = Copies of the 3rd party library DLLs (for library reference)

The Common folder contains copies of the 3rd party library DLLs so that the Utility
library projects all have a single folder to reference for the 3rd party DLLs on 
which they depend.
