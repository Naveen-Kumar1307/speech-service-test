EduSpeak Autopron Models

This directory contains the autopron models for the EduSpeak Toolkit.

  • ES_EnglishAdults: Used with English adult master packages english-*16-*.
  • ES_EnglishAdultsUK: Used with English adult master packages english-uk-*16-*.
  • pron3: Used with English adult master packages english-p8, english-g8.
These models may be redistributed with applications. However, you should not redistribute autopron models that are not used by your application.
