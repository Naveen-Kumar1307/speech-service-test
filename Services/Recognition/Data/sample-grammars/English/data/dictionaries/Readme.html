EduSpeak Dictionaries

This directory contains human readable dictionaries for each master package in the master_packages directory.

Each subdirectory corresponds to a single master package, and contains a human-readable ASCII file named DICTIONARY.ascii. These dictionaries can be viewed with any standard text editor, and are distributed only to provide example pronunciations to developers when generating pronunciations for new words.

There are actually only a few different dictionaries, as some master packages use the same dictionary.

All of these human-readable ASCII dictionaries are SRI proprietary.

EduSpeak Toolkit developers should never redistribute the ASCII format version of these dictionaries to users of their applications, or anyone else, without express written permission from SRI. They should also never be copied into the master package directories, as that might lead to accidental redistribution.
