Explanation of terms:

  • Master Package Name:  This is the name of an EduSpeak master package.  The master package is used as an argument for the "eduspeak-compile" program.  You will also need to match the scoring package to the master package, as described in the Scoring Readme file.
  • Language:  The language this master package is designed to recognize.
    • English:  North American English.
  • Age:  The speaker age range this master package is optimized to recognize.  For example, "english-p16-na-991221-v2" is designed to recognize adult speakers only.
  • Gender:  The speaker gender this master package is optimized to recognize.  For example, some gender-dependent master packages have been trained to recognize only the speech of male speakers (e.g., "english-p16-na-m-980909-v2"), and will perform poorly for females, or vice-versa ("english-p16-na-f-980909-v2").  Other gender-independant master packages have been optimized for both male and female speakers ("english-p16-na-991221-v2").  Because of the performance loss associated with using male models for female speakers (or vice-versa), for best results SRI recommends that applications use only gender-independent master packages unless it can be guaranteed that every speaker can always be recognized with the appropriate gender-dependent model.
  • Speaker Type:  "Native" models are optimized only for recognizing native speakers of a language.  "Non-native" models have been adapted to be good at recognizing both native and non-native speakers of a language. 
  • Model Type:  The types of master package are PTM, PSTM, and Genone.
    • Genone models are usually more accurate, but use more resources (disk space, RAM, CPU cycles) than PTM models.
    • PTM models are more compact and recognize speech faster, but at the expense of slightly worse recognition performance.
    • PSTM models are in between PTM and Genone models in size and performance.
  • Scoring Available:  Whether pronunciation scoring is available with this master package.  For more information, see the Scoring Readme file.

Notes:

  1. english-p16-na-nn-m-000105-v2, english-p16-na-nn-f-000105-v2:  Use for Japanese non-native adult speakers of English.
  2. english-p16-na-050930-v2:  This model uses normalized acoustic features, which is supported only by the EduSpeak Toolkit version 2.20 or better.
  3. english-uk-s16-na-080313:  This model was trained by adapting an American English model to British speakers, and also by modifying an American dictionary.
  4. english-p8, english-g8: Do not use these models in applications, they are very old and are less accurate. Unlike the rest of the EduSpeak models, these use a 8KHz sampling rate rather than 16KHz.
