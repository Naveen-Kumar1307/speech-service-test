EduSpeak Speech Recognizer Data

This directory contains data necessary to run or compile applications with the EduSpeak Speech Recognition Toolkit. Please contact EduSpeak if you have any questions about the contents of this directory.

The data directory contains the following files and directories:

  • dictionaries: This directory contains human-readable versions of the EduSpeak master dictionaries. All of these human-readable ASCII dictionaries are SRI proprietary. EduSpeak Toolkit developers should never redistribute these dictionaries to users of their applications, or anyone else, without express written permission from SRI. See the Readme file for more information.

  • license-codes: This file specifies whether the recognizer and associated tools can be run on your computer.

  • master_packages: This directory contains the EduSpeak acoustic models, which are used for recognizing speech. Read this.

  • nuance-resources.defaults: This file specifies the default value of many EduSpeak parameters.

  • pron: This directory contains models for generating pronunciations for new words which can't be found in the system dictionaries. Read this.

  • scoring: This directory contains the EduSpeak scoring models, which are used for scoring speech pronunciation. Read this.
